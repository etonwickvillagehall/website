Clearing your browsers cache

If you saw two green ticks on the previous page and you are still seeing a text only version of the site please clear your web browser’s cache, return to the site’s homepage and refresh the browser.

Step One

If you’re not reading this in your web browser then you’ll need to open it. There are quite a few web browsers available, we’ll cover the basics on the most popular ones.

Internet Explorer 6

  1. Open the Tools menu and select Internet Options.
  2. If the general tab is not selected, select it.
  3. Select Delete Files in the Temporary Internet Files section and choose OK.

Internet Explorer 7

  1. Open the Tools menu and select Internet Options.
  2. If the general tab is not selected, select it.
  3. Select Delete… from the Browsing history section.
  4. Select Delete files… from the Temporary Internet Files section and choose OK.

Firefox 1.0/1.5

  1. Open the Tools menu and select Options.
  2. Choose the Privacy icon and then the Cache tab.
  3. Select Clear Cache Now.

Firefox 2.0

  1. Open the Tools menu and select Options.
  2. Choose the Advanced icon and then the Network tab.
  3. Select Clear Now from the Cache section.

Mac users will need to open the Preferences dialog, listed under the Firefox menu and then follow the instructions from step 2.

Safari 2.0/3.0 OS X

  1. Open the Safari menu and select Empty Cache….
  2. Select Empty from the dialog box.

Windows users will need to open the Edit menu, select Empty Cache… and choose Empty.

Opera 9

  1. Open the Tools menu and select Preferences….
  2. Select the Advanced tab and then the History item from the list to the left.
  3. Select Empty now.

Mac users will need to open the Opera menu, select Preferences… and then follow the instructions from step 2.

AOL

  1. Open the Tools menu and select Internet Options.
  2. If the general tab is not selected, select it.
  3. Select Delete Files in the Temporary Internet Files section
  4. Select the Delete all offline content box to place a check mark in it then choose the OK button.
  5. When refreshing the site homepage make sure you press CTRL+F5 instead of using the refresh button.
